# Opage

> Homepage for your browser

`Opage` 是一个客制化上网首页

基于 Quasar & Vue.js 第2次重构

集成搜索，网站，天气，TODO

Faster & Pure


## 功能   

- 搜索：Google + Bing + 百度 + 搜狗  (可自定义更多搜索引擎)
- 网站：可自定义网站列表
- ~~天气：全国城市~~ (目前先取消，后期更新更好的天气接口,闲暇时间更新)
- TODO: 简单的 Todo 列表
- 完整的数据备份及恢复
